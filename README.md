# B K Agarwal & Co | CA Firm Landing Page

A modern, responsive, and animated hero section and landing page prototype built for B K Agarwal & Co, a Premium Chartered Accountant firm based in Bangalore.

## 🚀 Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Motion (Framer Motion)](https://motion.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🛠️ Local Development

Follow these steps to run the project locally on your machine:

1. **Clone the repository** (or download the folder):
   ```bash
   git clone <your-repo-url>
   cd bk-agarwal-ca

2. **Install dependencies:**

   ```Bash
   npm install

3. **Start the development server:**

   ```Bash
   npm run dev
   
The app will typically be available at http://localhost:3000 or http://localhost:5173.
